JSON çıktı kuralları:
- Yalnızca tek bir geçerli JSON nesnesi veya istenen durumda JSON dizi üret.
- Markdown kod bloğu, açıklama, önsöz veya sonsöz ekleme.
- Alan adlarını değiştirme, yeni üst seviye anahtar ekleme.
- Tip disiplinini koru: boolean alanlar true veya false, sayılar sayısal, listeler liste olarak gelsin.
