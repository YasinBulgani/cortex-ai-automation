# Test Dünyası Web Sitesi

Test dünyası hakkında bilgilendirici, modern ve responsive bir web sitesi.

## Özellikler

- 🎨 Modern ve şık tasarım
- 📱 Tam responsive (mobil uyumlu)
- ⚡ Hızlı ve optimize edilmiş
- 🎯 Smooth scroll navigasyon
- ✨ Animasyonlu geçişler
- 📊 İstatistik sayaçları
- 📝 İletişim formu

## Kullanım

Web sitesini görüntülemek için `index.html` dosyasını bir web tarayıcısında açın.

### Yerel Sunucu ile Çalıştırma

```bash
# Python 3 ile
python3 -m http.server 8000

# Node.js ile (http-server kurulu ise)
npx http-server

# PHP ile
php -S localhost:8000
```

Ardından tarayıcınızda `http://localhost:8000` adresine gidin.

## İçerik

Web sitesi aşağıdaki bölümleri içerir:

1. **Ana Sayfa (Hero)**: Hoş geldiniz mesajı ve istatistikler
2. **Hakkında**: Test dünyası hakkında genel bilgiler
3. **Test Türleri**: Farklı test türleri ve açıklamaları
4. **Araçlar**: Popüler test otomasyon araçları
5. **En İyi Uygulamalar**: Test otomasyonunda best practices
6. **İletişim**: İletişim formu ve bilgileri

## Dosya Yapısı

```
website/
├── index.html      # Ana HTML dosyası
├── styles.css      # CSS stilleri
├── script.js       # JavaScript fonksiyonları
└── README.md       # Bu dosya
```

## Özelleştirme

### Renkleri Değiştirme

`styles.css` dosyasındaki `:root` değişkenlerini düzenleyerek renkleri özelleştirebilirsiniz:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### İçerik Düzenleme

`index.html` dosyasını açarak içerikleri düzenleyebilirsiniz.
